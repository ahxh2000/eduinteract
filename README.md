工具管理功能按以下步骤逐步实现：
第一阶段：基础工具管理界面
1. 管理员登录和权限控制
管理员登录页面
权限验证中间件
管理后台布局（侧边栏导航）
2. 工具列表页面
显示所有工具的基础信息（标题、学科、上传时间、状态）
分页功能
基础筛选（按学科、状态）
表格形式展示，支持排序
第二阶段：工具操作功能
3. 工具详情查看
点击工具查看详细信息
显示工具的所有字段（标题、描述、文件URL、上传者等）
预览工具文件（iframe嵌入）
4. 工具状态管理
审核功能（通过/拒绝/下架）
状态变更操作
审核意见填写
状态变更历史记录
5. 工具编辑功能
修改工具基本信息（标题、描述、学科）
表单验证
编辑历史记录
第三阶段：高级管理功能
6. 批量操作
批量选择工具
批量审核
批量删除
批量修改状态
7. 高级筛选和搜索
按上传时间范围筛选
按上传者筛选
关键词搜索
多条件组合筛选
8. 工具删除功能
单个工具删除
删除确认对话框
删除后处理（是否保留文件）
第四阶段：数据统计和监控
9. 管理统计面板
工具总数统计
各状态工具数量
各学科工具分布
今日新增工具数
10. 操作日志
记录所有管理操作
操作历史查看
操作者追踪
具体实现步骤建议：
步骤1：创建管理后台基础结构
创建 /admin 路由

创建工具列表组件

步骤3：工具详情和状态管理
创建工具详情页面
实现状态变更功能
添加审核意见功能
步骤4：工具编辑功能
创建工具编辑表单
实现数据更新功能
添加表单验证
步骤5：批量操作
实现多选功能
创建批量操作组件
实现批量处理逻辑
步骤6：高级功能

添加统计面板
